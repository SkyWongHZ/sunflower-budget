# 记账本接口文档

除登录接口外,所有接口都需要在请求头中携带 token
```javascript
Headers: {
  "Authorization": "Bearer ${token}"  // 登录接口返回的 token
}
```

## 用户模块

### 用户登录
```javascript
POST /api/auth/login

请求参数:
{
  "username": string, // 用户名
  "password": string  // 密码
}

请求示例:
{
  "username": "zhangsan",
  "password": "123456"
}

返回数据:
{
  "code": 200,        // 状态码
  "msg": "success",   // 提示信息
  "data": {
    "token": string,  // 登录令牌
    "user": {
      "id": string,
      "username": string
    }
  }
}

返回示例:
{
  "code": 200,
  "msg": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "id": "65c3724e6c3c0ef1c8a65432",
      "username": "zhangsan"
    }
  }
}
```

## 标签模块

### 预设标签说明
预设标签是系统提供的通用标签，具有以下特点：
1. 预设标签对所有用户可见，作为共享的默认标签使用
2. 普通标签是用户个人创建的标签，仅对创建者可见
3. 标签类型通过 `tagType` 字段区分：`preset`(预设标签) 和 `custom`(用户自定义标签)
4. 预设标签适用于常见的收支类别，如"餐饮"、"交通"、"工资"等
5. 当前版本暂未启用RBAC权限控制，后续版本将加入角色权限管理

### 查询标签列表
```javascript
GET /api/tags

请求参数(Query参数):
{
  "type": string,      // 可选 标签类型 expense(支出)/income(收入)
  "pageIndex": number, // 可选 默认1
  "pageSize": number   // 可选 默认20
}

请求示例:
/api/tags?type=expense&pageIndex=1&pageSize=10

返回数据:
{
  "code": 200,         // 状态码
  "msg": "success",    // 提示信息
  "data": {
    "total": number,   // 总条数
    "list": [{
      "id": string,    // 标签唯一id
      "name": string,  // 标签名称
      "icon": string,  // 标签图标
      "type": string   // 标签类型,支出和收入
      "tagType": string, // 预设标签类型 preset(预设)/custom(自定义)
    }]
  }
}

返回示例:
{
  "code": 200,
  "msg": "success",
  "data": {
    "total": 2,
    "list": [
      {
        "id": "65c3724e6c3c0ef1c8a65433",
        "name": "餐饮",
        "icon": "food",
        "type": "expense",
        "tagType": "preset"
      },
      {
        "id": "65c3724e6c3c0ef1c8a65434",
        "name": "交通",
        "icon": "car",
        "type": "expense",
        "tagType": "custom"
      }
    ]
  }
}
```
