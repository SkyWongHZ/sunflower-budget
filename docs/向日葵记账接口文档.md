# 记账本接口文档

除登录接口外,所有接口都需要在请求头中携带 token
```javascript
Headers: {
  "Authorization": "Bearer ${token}"  // 登录接口返回的 token
}
```

## 用户模块

### 用户登录
```javascript
POST /api/auth/login

请求参数:
{
  "username": string, // 用户名
  "password": string  // 密码
}

请求示例:
{
  "username": "zhangsan",
  "password": "123456"
}

返回数据:
{
  "code": 200,        // 状态码
  "msg": "success",   // 提示信息
  "data": {
    "token": string,  // 登录令牌
    "user": {
      "id": string,
      "username": string
    }
  }
}

返回示例:
{
  "code": 200,
  "msg": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "id": "65c3724e6c3c0ef1c8a65432",
      "username": "zhangsan"
    }
  }
}
```

## 标签模块

### 查询标签列表
```javascript
GET /api/tags

请求参数(Query参数):
{
  "type": string,      // 可选 标签类型 expense(支出)/income(收入)
  "pageIndex": number, // 可选 默认1
  "pageSize": number   // 可选 默认20
}

请求示例:
/api/tags?type=expense&pageIndex=1&pageSize=10

返回数据:
{
  "code": 200,         // 状态码
  "msg": "success",    // 提示信息
  "data": {
    "total": number,   // 总条数
    "list": [{
      "id": string,    // 标签唯一id
      "name": string,  // 标签名称
      "icon": string,  // 标签图标
      "type": string   // 标签类型,支出和收入
      "tagType": string, // 预设标签类型 preset(预设)/custom(自定义)
    }]
  }
}

返回示例:
{
  "code": 200,
  "msg": "success",
  "data": {
    "total": 2,
    "list": [
      {
        "id": "65c3724e6c3c0ef1c8a65433",
        "name": "餐饮",
        "icon": "food",
        "type": "expense",
        "tagType": "preset"
      },
      {
        "id": "65c3724e6c3c0ef1c8a65434",
        "name": "交通",
        "icon": "car",
        "type": "expense",
        "tagType": "custom"
      }
    ]
  }
}
```

  "endDate": string,    // 可选 结束日期 YYYY-MM-DD
  "type": string        // 可选 'income' | 'expense'

## 文件上传模块

### 上传文件到 MinIO
```javascript
POST /api/minio/upload

请求格式:
Content-Type: multipart/form-data

请求参数:
{
  "file": File  // 文件对象，支持图片等格式
}

返回数据:
{
  "code": 200,           // 状态码
  "msg": "success",      // 提示信息
  "data": {
    "url": string        // 文件访问地址
  }
}

返回示例:
{
  "code": 200,
  "msg": "success",
  "data": {
    "url": "http://118.178.184.13:9000/sunflower-budget/1742975079930-example.jpg"
  }
}

注意事项:
1. 文件上传接口支持多种格式的文件，主要用于上传账单凭证图片
2. 返回的 URL 可直接用于图片展示或下载
3. 文件名会自动添加时间戳前缀，确保唯一性
4. 上传的文件会存储在名为 "sunflower-budget" 的存储桶中
```
